# 注意

这个脚本使用GPLv3许可协议

https://www.lanqiao.cn/courses/115

目前只支持实验三《系统调用》

# 介绍

这是一个我在学习shell编程时写的玩具项目，也是我在Github上传的第一个项目

它不会帮你写实验报告。实现这个功能非常的困难。

但是，它可以帮你自动初始化环境，而不是每一次启动环境都要敲一串麻烦又意义不大的代码。并且，它可以自动写代码，编译环境。你需要做的只有截图以及完成实验报告
